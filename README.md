# lifeground-satellite

Satellite data integration (Sentinel-2/Landsat)

## Part of LifeGround Ecosystem

This repository is part of [LifeGround](https://github.com/LifeGround/lifeground-core) - an open-source platform to revive infertile soils worldwide.

## Quick Start

```bash
# Clone entire ecosystem with submodules
git clone --recursive https://github.com/LifeGround/lifeground-core.git
cd lifeground-core/repos/lifeground-satellite

# Or clone just this repo
git clone https://github.com/LifeGround/lifeground-satellite.git
```
